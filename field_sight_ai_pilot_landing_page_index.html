Automated Root Cause Analysis for Industrial & Service Equipment

FieldSight AI turns field notes, telemetry, and evidence into a deterministic RCA report. Built around a schema‑first API with an Evidence Safety System (ESS) gate for high‑integrity outputs.

Why teams choose FieldSight

Deterministic where it matters. Composable where it counts.

Schema‑first API

Versioned JSON schemas for auditability. Backwards‑compatible minor releases keep integrations stable.

Deterministic Reasoning

Strict mode sets temperature = 0 for reproducible results. Trace IDs and spans for every stage.

Evidence Safety System (ESS)

Confidence gating enforces high‑integrity outputs before results are accepted or escalated.

Composability

Use stages independently (Interpret, Analyze, Iterate) or orchestrate via Diagnose.

Reporting

Generate branded, shareable PDF/HTML reports from canonical RCA results with localization and redaction.

Security & Rate Limits

X‑Api‑Key required. MIME allowlists. EXIF stripping. 10 requests/sec per org by default.

How it works

1

Interpret

2

Analyze

Fuse evidence deterministically to produce a ranked hypothesis set with ESS.

3

Iterate

Refine the RCA with follow‑ups or new evidence without starting over.

4

Diagnose & Report

One‑click orchestrator plus branded PDF/HTML output for customers and archives.

API at a glance

Endpoints mirror the reasoning flow — minimal integration, maximum clarity.

Endpoint Purpose Notes
POST /v1/interpret NL/evidence → Context Frame Deterministic mode for auditability
POST /v1/analyze Context Frame → RCA envelope ESS confidence gating
POST /v1/iterate Update RCA with new evidence Keeps trace IDs and spans
POST /v1/diagnose Orchestrate interpret + analyze Latency budgets, cache hints
POST /v1/report Branded PDF/HTML report Presigned URLs, localization
